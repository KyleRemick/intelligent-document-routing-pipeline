# Sample documents

This folder is for **example materials** you use when demoing or testing.

- Prefer **synthetic PDFs** you create locally (no real patient data).
- Example naming: `referral_001.pdf`, `lab_result_sample.pdf`, `insurance_card_demo.pdf`.
- If you do not commit binary PDFs, describe here what you used during interviews (e.g. “one-page referral PDF with the words *referral*, *specialist*, *consult*”).

For automated tests, the repository uses **unit tests** on classification strings instead of large PDF fixtures.
